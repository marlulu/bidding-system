# 内部招标网站系统 - 项目交付说明

## 🚀 项目概述
本项目是一个基于 **Spring Boot 3.x** 和 **Vue 3** 开发的专业级内部招标采购门户。系统采用深蓝色调设计，体现专业与信任，支持 PC 和移动端响应式布局。

## 🛠 核心交互设计

### 1. 弹窗登录（无独立登录页）
为了提升用户体验，系统**彻底移除了独立的登录页面**。
- **触发方式**：点击顶部导航栏右侧的“登录”按钮，或在未登录状态下尝试访问受限内容（如公告详情）。
- **交互效果**：系统会弹出简洁的登录对话框，登录成功后自动刷新当前页面状态或重定向至目标页面。

### 2. 登录后状态切换
- **头像显示**：登录成功后，原有的“登录/注册”按钮将替换为用户头像。
- **下拉菜单**：鼠标悬停在头像上，可展开下拉菜单，包含“个人中心”、“我的收藏”及“退出登录”。

### 3. 免登录浏览
- **公开内容**：首页、招标公告列表、供应商名录、政策法规列表均支持免登录浏览。
- **受限内容**：仅在查看具体详情、下载附件或参与投标时才强制要求登录。
